FONTLOG for the Apfel Grotezk font

This file provides detailed information on the Apfel Grotezk Font Software.
This information should be distributed along with the Apfel Grotezk font
and any derivative works.

Basic Font Information

Apfel Grotezk is a round and airy sans serif family, inspired by neo-grotesque and geometric typefaces.

To contribute to the project contact Collletttivo
at collletttivo@gmail.com
